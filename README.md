# Matplotlib Data Visualization Reference Guide

Welcome to the **Matplotlib Data Visualization Reference Guide**! This project serves as a clean, structured gallery and implementation reference for the most essential data visualization plot types using Python's `matplotlib.pyplot` and `numpy` libraries. 

It is designed to showcase how to write clean plotting scripts in Jupyter Notebooks and pair them directly with their corresponding data outputs.

## 📊 Visualizations Covered

The repository features 6 key plotting techniques, complete with clean formatting, titles, axis labels, legends, and styling configurations:

1. **Line Chart** – Customization with specific data markers, dash styles, label definitions, and structural grids to display progression over time.
2. **Bar Chart** – Categorical data presentation utilizing distinct color palettes and edge borders to contrast values across independent groups.
3. **Pie Chart** – Proportional distribution layouts using slice percentages, drop shadows, start-angle configurations, and exploded slice emphasis.
4. **Histogram** – Frequency distribution visualization modeling large continuous datasets into discrete value bins.
5. **Scatter Plot** – Correlation and distribution modeling across two continuous variables to inspect data trends and densities.
6. **Subplots** – Multi-panel layouts (`nrows=1, ncols=2`) demonstrating how to organize and render multiple distinct mathematical wave functions in a single unified figure.
